# Only_Won

* 프로젝트 이름 : `Only Won`  
* 장르 : `캐주얼` / `경쟁` / `전략` / `멀티` / `탑뷰` / `콘솔`  
* 개발 환경 : `Visual Studio 2022` / `DirectX 2D` / `C++`  
* 제작 기간 : `2023.08.07` ~ `2023.08.25`  
* 수상 내역 : `최우수상` 🏆
* 담당 파트 :
  1. `충돌 콜라이더 및 물리 효과 구현`
     - 간단한 Rigidbody, 가속도를 이용한 부드러운 움직임
     - boxCollider, circleCollider, AABB, OBB
  2. `특정 콜라이더와 부딪힐 수 있는 레이어 시스템`
  3. `각종 매니저 구현`
     - TimeManager
     - WorldManager
     - InputManager
     - CollisionManager
  4. `Object 계층 구조 설계`
      - 유니티의 계층 구조를 참고
      - Object, Transform, ..
  5. `플레이어 FSM`
      - Idle, Run, Stun, Die
  6. `기차 시스템 및 패턴 제작`
  7. `xInput API를 이용한 Xbox 컨트롤러 사용`
     - 4개의 컨트롤러 동시 입력
  8. `아이템의 기능 및 연출 이펙트`
  9. `각종 유틸 및 콘텐츠 제작`
     - Math Class, Define ..
